Create array comparator function.

Parameters
options (object?)
Name Description
options.numeric boolean (default false) Sort numerically. Defaults to lexicographic/alphabetic sort.
options.descending boolean (default false) Sort in descending order. Defaults to ascending order.
options.array boolean (default false) Sort arrays. Nested arrays are also compared recursively.
options.by (number | string | getter) (default undefined) Sort by either array index, a callback(element): any - or by object keys with dot-notation support.

sortNumeric

Numerical comparison of items. If the passed objects are not numbers, their .valueOf() methods are called to retrieve a numeric value representation of them.

sortNumeric(a: (number | object), b: (number | object)): number
Parameters
a ((number | object)) First item to compare.
b ((number | object)) Second item to compare.
Returns
number: A positive number if a > b, a negative number if a < b, 0 otherwise.

Parameters
